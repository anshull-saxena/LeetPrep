**Title:** Show HN: LeetPrep – Open-source LeetCode practice app with built-in code editor

**Post:**

https://hopefuel.vercel.app

I built a free, open-source LeetCode practice tool. Key features:

- Company-wise questions from 100+ companies with frequency data
- Blind 75/150/300 curated paths
- Monaco code editor with in-browser JavaScript/TypeScript execution (no backend needed)
- Full problem descriptions fetched live from LeetCode's GraphQL API
- Progress tracking with Google Sign-In + Supabase (local-first sync)
- Question-specific starter code with proper function signatures

Built with Next.js 15, TypeScript, Tailwind. ~2.5k lines of TSX.

GitHub: https://github.com/anshull-saxena/LeetPrep
