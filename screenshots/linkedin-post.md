I built LeetPrep — a free, open-source LeetCode practice app — and I'm excited to share it with the community! 🚀

**What makes it different:**
• Practice by company — see exactly what Google, Amazon, Meta, and 100+ other companies ask
• Follow Blind 75/150/300 study paths with progress tracking
• Built-in code editor (Monaco/VS Code engine) — write and run JavaScript/TypeScript right in the browser
• Full problem descriptions with test cases and hints — fetched live from LeetCode
• Sync your progress across devices with Google Sign-In

All free, no paywalls, no sign-up required to browse.

Try it: https://hopefuel.vercel.app
Repo: https://github.com/anshull-saxena/LeetPrep

Built with Next.js 15, TypeScript, Tailwind CSS, Firebase Auth, and Supabase.
