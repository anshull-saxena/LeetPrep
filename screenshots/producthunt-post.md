# Product Hunt

**Tagline:** Practice LeetCode questions company-wise or via Blind 75/150/300 — with a built-in code editor

**Description:**

LeetPrep is a free, open-source LeetCode practice platform that brings everything into one place:

**📊 Company-Wise Practice**
Browse questions from 100+ companies (Google, Amazon, Meta, Microsoft, Apple, Netflix, and more) sorted by frequency. Filter by difficulty and time period.

**📚 Blind 75 / 150 / 300 Paths**
Curated study paths with topic filters, difficulty filters, and progress tracking.

**⌨️ Built-in Code Editor**
Monaco Editor (VS Code engine) with 8 languages. JavaScript and TypeScript execute instantly in the browser — no backend needed.

**📖 Problem Descriptions**
Click any question → full LeetCode description, test cases, and hints.

**☁️ Cross-Device Sync**
Sign in with Google. Progress syncs via Supabase. Works offline.

**Tech Stack:** Next.js 15 · TypeScript · Tailwind · Firebase Auth · Supabase · Monaco Editor

**Price:** Free (open source, MIT)
